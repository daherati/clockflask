# clockflask

This is a simple clock application built using Python and Flask that displays the current time.

## Prerequisites

- Python 3.x
- pip
- flask

# Installation:

## Install Flask

`pip install flask
`



1. Clone the repository:
   

`git clone https://github.com/daherati/clockflask
`
Create virtual environment:

`python3 -m venv venv
`
